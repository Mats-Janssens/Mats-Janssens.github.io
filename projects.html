    • card dealer robot

      A third semester engineering project. We were free to do whatever we wanted as long as we were detecting cards and recognising hand gestures. We decided to build a robot as to make the game more interactive.

      The robot can play blackjack and present the game on the screen. It's quite easy to add new games but we did not have time. The robot gives the different players a card. It reads in which card was dispensed using a small camera.

      Then it asks the player for a bet. The player enters the bet and makes the ok sign (ðŸ‘Œ). It repeats this for all the players. Then the game starts and the players can do different things based on their hand symbol.
